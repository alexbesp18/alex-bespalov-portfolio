# Miner Price Analytics

Web scraping tools and data for tracking Bitcoin miner hardware prices from marketplaces.

## Features

- Jupyter notebook for scraping OneMiners.com marketplace
- Historical price data snapshots
- Excel data exports

## Files

- `scrape_oneminers.ipynb` - Web scraping notebook
- `08-03-25.xlsx` - Price data snapshot (August 3, 2025)
- `08-04-25.xlsx` - Price data snapshot (August 4, 2025)

## Usage

Open the Jupyter notebook:
```bash
jupyter notebook scrape_oneminers.ipynb
```

## Tech Stack

- Python
- Jupyter
- Pandas
- Web scraping libraries
