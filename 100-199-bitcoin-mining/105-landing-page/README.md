# Crypto Calculators Landing Page

Marketing and navigation hub for all Bitcoin mining calculators.

## Features

- Links to all deployed calculators
- Calculator descriptions and use cases
- Donation information (Strike, BTC address)

## Live Demo

https://btc-calculators.vercel.app

## Linked Calculators

- Bitcoin Mining Calculator
- Bitcoin Mining Profitability Calculator
- Miner Acquisition Calculator
- BTC Loan Calculator
- Hosted Mining Portfolio Manager
- Miner Price Tracker

## Tech Stack

- React 19
- Lucide Icons

## Setup

```bash
npm install
npm start
```
