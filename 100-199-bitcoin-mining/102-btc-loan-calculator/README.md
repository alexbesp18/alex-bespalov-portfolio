# BTC Loan Calculator

Bitcoin-backed loan analysis tool.

## Features

- LTV (Loan-to-Value) monitoring
- Liquidation price calculations
- Interest cost projections
- Price shock scenario analysis
- Future growth projections
- Borrowing power calculator

## Live Demo

https://btc-loan-calculator.vercel.app

## Use Case

"How much BTC do I need as collateral for a $50k loan?"

## Tech Stack

- React 19
- Lucide Icons
- LocalStorage for data persistence

## Setup

```bash
npm install
npm start
```
