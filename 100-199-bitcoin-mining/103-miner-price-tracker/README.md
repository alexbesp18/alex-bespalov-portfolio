# Miner Price Tracker

> **Related**: See [miner-price-analytics](../miner-price-analytics) for the data scraping tools used to populate this data.

Track historical miner hardware prices over time.

## Features

- Upload Excel/CSV price data
- Daily and intraday price tracking
- Price history charts
- Rollback functionality for data management
- Efficiency database for auto-fill
- Storage optimization

## Live Demo

https://miner-price-tracker.vercel.app

## Use Case

"Track miner price trends over time to find the best time to buy"

## Tech Stack

- React 19
- Recharts (for charts)
- Papa Parse (CSV parsing)
- XLSX (Excel parsing)
- Lucide Icons
- LocalStorage for data persistence

## Setup

```bash
npm install
npm start
```
