# AA Tools

American Airlines points optimization tools monorepo.

## Projects

| Folder | Description |
|--------|-------------|
| 001-aa-scraper | AA Points Arbitrage Monitor - automated SimplyMiles + Portal stacking alerts |
| 002-aa-streak-optimizer | AA Hotel Streak Optimizer - Find optimal points-per-dollar hotel sequences |
| 003-quick-aa-hotels | Quick AA Hotels search interface |
| 004-us-hotel-scraper | US Hotel Scraper with city discovery and scoring |

## Structure

Each project maintains its own:
- `CLAUDE.md` - Project-specific Claude memory
- `requirements.txt` or `package.json` - Dependencies
- `README.md` - Project documentation
